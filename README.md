# Проект Демонстрационного экзамена по МДК.02.02

## Описание проекта

Данный репозиторий содержит материалы проекта по модулю **МДК.02.02**. В проекте представлены файл с комментариями и скрипт для развертывания базы данных.

## Состав репозитория

В репозитории находятся файлы:

*   **`DEMO_0202_xml.xml`**: Файл содержит комментарии проекта.
*   **`bd_demo.sql`**: Скрипт базы данных.
*   **`Er_диарграмма_базыДанных.pdf`**: ER-диаграмма базы данных в .pdf.
